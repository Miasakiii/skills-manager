## 功能

通过多轮问答引导用户完成面试准备：

1. **职位分析**：收集目标职位信息，分析关键要求
2. **技能匹配**：匹配用户技能与职位需求
3. **问题准备**：生成常见面试问题及参考答案
4. **模拟练习**：提供模拟面试场景

## 对话流程

### 第一阶段：信息收集

**问题 1**：目标职位是什么？

```
请输入目标职位名称和公司（如有）：
示例：字节跳动 - 高级前端工程师
```

**问题 2**：工作年限？

```
请输入您的工作年限：
- 应届生
- 1-3 年
- 3-5 年
- 5 年以上
```

**问题 3**：核心技术栈？

```
请列出您熟悉的技术（逗号分隔）：
示例：React, TypeScript, Node.js, Python
```

### 第二阶段：分析与准备

根据收集的信息，生成：

- 职位要求分析
- 技能匹配度评估
- 推荐准备的重点领域

### 第三阶段：问题练习

提供三类常见问题：

1. **技术问题**：基于职位要求的技术问题
2. **行为问题**：STAR 法则相关问题
3. **系统设计**：架构设计类问题（高级职位）

## 示例对话

```
用户：我想准备面试
助手：好的，我来帮您准备。请问您的目标职位是什么？

用户：腾讯的后端工程师
助手：明白了。请问您的工作年限是多少？

用户：3年
助手：了解。请列出您熟悉的技术栈：

用户：Java, Spring Boot, MySQL, Redis
助手：很好！基于您的情况，我为您准备了面试方案...
```


## 适用场景

- 求职者准备面试
- 转行者了解目标职位要求
- 在职者准备晋升答辩

## 不适用

- 简历撰写（使用专门的简历工具）
- 薪资谈判（需要具体市场数据）
- 职业规划咨询（需要长期跟踪）
